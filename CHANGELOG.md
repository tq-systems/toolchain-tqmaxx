## 2.0.0

### Added

* bare-base: new tools for U-Boot EFI and OP-TEE
* bare-base: add git-lfs to deal with NXP MCU SDK blobs stored in LFS repo.
* Dockerfile*: add support for yocto based on ubuntu 22.04
* Makefile: add support for containers based on Ubuntu 22.04
* Add support for containers for TI Code Composer Studio
* base: add container based o Ubuntu 22.04
* base/Dockerfile.ubuntu-22.04: add libc i386 packages
* yocto-base/Dockerfile: add crudini and python3-cairo
* base: add packages needed by checkpatch
* base: enable git-lfs
* add python container
* bare-base: add yamllint for modern U-Boot binman support
* base/bare/yocto: add containers based on Ubuntu 24.04
* ptxdist: ubuntu-14.04: add buildman config

### Changed

* ptxdist/yocto: update ci container to use openjdk 17  
* base: improve ppa handling using add-apt-repository
* base: use git from ppa for Ubuntu 14.04/16.04 
* Makefile: switch to docker-compose-v2
* use up-to-date lavacli
* base: enable extra gitconfigs for ci builds
* remove limits from base gitconfig
* use container specific buildman configs
* Makefile: do not build outdated containers with all target

### Fixed

* base/Dockerfile.ubuntu-14.04: fix apt update error
* base: fix sources.list mirror URL for de.archive.ubuntu.com
* base: fix deprecated ENV syntax without assignment operator
* treewide: fix ARG without default value to suppress build warnings
* base: move certificate handling before first apt-get
* bare: fix buildman config for ubuntu-24.04
* scripts/usermap.sh: fix wrong variable used to set primary group
* base/Dockerfile.ubuntu-14.04: build git against OpenSSL for TLS 1.2 compatibility
